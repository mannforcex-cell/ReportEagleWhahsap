## 📋 **Overview**

WhatsApp Report Generator adalah script yang membolehkan anda mengeksport perbualan WhatsApp dan menjana laporan dalam format yang kemas dan profesional. Sesuai untuk kegunaan rekod, bukti perbualan, atau dokumentasi rasmi.

---

## ✨ **Features**

| # | Feature | Description |
|:-:|---------|-------------|
| 1 | 📊 **Export Chat** | Eksport perbualan individu atau kumpulan |
| 2 | 📄 **Multiple Format** | Output dalam PDF, HTML, atau TXT |
| 3 | 🔍 **Filter & Search** | Cari mesej berdasarkan kata kunci, tarikh, atau pengirim |
| 4 | 📈 **Statistik** | Analisis perbualan - jumlah mesej, pengirim paling aktif, waktu puncak |
| 5 | 🖼️ **Media Support** | Papar gambar, video dan dokumen dalam laporan |
| 6 | 🔐 **Private Mode** | Hasilkan laporan tanpa simpan data di server |
